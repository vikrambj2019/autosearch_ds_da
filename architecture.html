3 · 2-Pass Deterministic Analytics Engine

Every number is computed by Python — the LLM only writes narrative from pre-computed JSON results including a pre-computed direction field.

sequenceDiagram participant U as User Question participant O as Orchestrator participant P as Param Extractor LLM participant A as analytics.py Python participant N as Narrator LLM participant Out as Output File U->>O: question + CSV path O->>O: profile data, detect grain O->>P: extract columns, periods, filters P-->>O: resolved params Note over O,A: Pass 1 - Primary Analysis O->>A: call primary function A-->>O: JSON result with direction field Note over O,A: Pass 2 - Depth Analyses, rule-based selection O->>A: depth analysis 1 A-->>O: JSON result O->>A: depth analysis 2 A-->>O: JSON result O->>N: all JSON results + company context N-->>O: English narrative report O->>Out: save timestamped markdown

5 · ML Autoresearch Loop
